# Ethereum-Network-Activity-Analysis-December-2025

## Overview

This project analyzes Ethereum on-chain activity to understand network usage, transaction growth, and wallet participation trends.

Using SQL queries on blockchain data, the analysis explores how transaction volume and active addresses evolve over time.

## Objectives

The analysis aims to answer the following questions:

* How many transactions occur on Ethereum daily?

* How has transaction activity evolved over time?

* How many wallets actively interact with the network?

* What trends indicate network growth?

## Key Insights from the analysis include
  * In December 2025, Ethereum processed 51.5M transactions, averaging 1.6M per day, 
indicating sustained high network usage.
